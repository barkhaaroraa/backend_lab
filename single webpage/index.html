Getting Started with HTML

Introduction

Welcome to the world of web development! HTML, or HyperText Markup Language, is the backbone of every web page. It's the standard language used to create and design documents on the World Wide Web. Whether you're looking to build a personal blog, a portfolio, or even a full-fledged website, HTML is where it all begins. This guide will walk you through the basics of HTML and help you get started on your journey to becoming a web developer.

What is HTML

HTML stands for HyperText Markup Language. It is a markup language used to create the structure of web pages. HTML uses a system of tags to define elements like headings, paragraphs, links, images, and more. These elements are the building blocks of a webpage.

HTML Elements Reference

This page contains all of the basic HTML components with the tags used.

Main Root

<html> Represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.

Document Metadata

  • <base> Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.

  • <head> Specifies relationships between the current document and an external resource. This element is most commonly used to link to CSS but is also used to establish site icons (both "favicon" style icons and icons for the home screen and apps on mobile devices) among other things.

  • <link> Represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.

  • <meta> Represents metadata that cannot be represented by other HTML meta-related elements, like <base>, <link>, <script>, <style> and <title>.
